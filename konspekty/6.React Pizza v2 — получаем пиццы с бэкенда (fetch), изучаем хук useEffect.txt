mockapi

в реакте, когда в компоненте стейт меняется компонент должен перерисоваться

хук useEffect - получает функцию, которую будет вызывать если произойдет какой-то эффект

у каждого компонента есть состояние жизненного цикла:
- состояние когда компонент отрисовался component did mount
- component did update - компонент обновился и мы можем делать что-то если компонент обновился
- когда компонент исчезает со страницы
- component will unmount - компонент будет удален размонтирован
эти жизненные циклы мы можем отловить с помощью этого хука
